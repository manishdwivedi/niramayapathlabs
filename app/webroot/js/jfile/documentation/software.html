- Software -

JFileUpload is a client-side JAVA component (Applet and WebStart). Here are the requirements on client-side to make it works:

  • JRE 1.4 (or higher).
  • Browser (Internet Explorer 6.x or higher, Safari, Firefox, Opera, Chrome, Netscape ...)
    or Web Start console.
